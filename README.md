# Allergies Detect

Start frontend en backend samen vanuit de hoofdmap:

```powershell
npm run dev
```

Stop ze weer met:

```powershell
npm run dev:stop
```

De backend draait standaard op `http://localhost:3010`. De frontend start op `http://localhost:5173`, of op de eerstvolgende vrije Vite-poort als `5173` al bezet is.
